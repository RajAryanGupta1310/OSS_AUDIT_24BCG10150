Project Description
   In this project, I explored Git as an open-source software and used it as a base to understand and apply Linux shell      scripting concepts. The goal was to get hands-on experience with how system-level tasks can be automated using simple     scripts.

Scripts Overview
   1.System Identity Report
     This script shows basic system details like the kernel version, current user, and how long the system has been            running.
   2.Package Inspector
     It checks whether Git is installed on the system and displays important information related to it.
     Disk Permission Auditor
   3.This script helps in checking directory sizes and understanding file permissions, which is useful for system              monitoring.
   4.Log File Analyzer
     It reads log files and counts how many times specific keywords appear, helping in basic log analysis.
   5.Open Source Manifesto Generator
     This script generates a simple file where I express my thoughts and approach towards open-source software.
